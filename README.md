## Overview

**GetWell** is a full-stack web platform developed as part of a Bachelor’s Thesis at the University POLITEHNICA of Bucharest.  
It provides an integrated solution for managing hospital workflows - connecting **patients**, **doctors** and **administrators** in a unified digital environment.  
The project aims to eliminate discontinuities in the medical process and simplify administrative operations.

---

## Objectives

- Unify patient, doctor and hospital management into a single digital platform  
- Enable scheduling, prescription generation and treatment management from one interface  
- Improve data accessibility and communication between departments  
- Optimize existing healthcare workflows and reduce redundant manual operations  

---

## Key Features

- **Role-based access control** for patients, doctors and administrators  
- **Appointment scheduling** and medical record management  
- **Prescription generation** based on real-time pharmacy stock  
- **Authentication and authorization** via Firebase integrated with Spring Security  
- **Modern responsive interface** using React and Material UI  
- **Containerized architecture** using Docker for consistent and scalable deployment  

---

## System Architecture

The GetWell platform is built on an **N-tier architecture**, separating responsibilities across multiple layers:

- **Frontend:** React + Material UI  
- **Backend:** Spring Boot (Java 23)  
- **Security:** Spring Security + Firebase Authentication  
- **Database:** MySQL (JPA/Hibernate ORM)  
- **Deployment:** Docker containers

## Results
